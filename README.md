# Grocery_List_App
A simple grocery list app built with Vue.js that allows users to add, remove, and check off grocery items.

This is a simple grocery list application built using **Vue.js** and **Font Awesome** via CDN links.  
It allows users to:

- Add grocery items with a quantity  
- Check off completed items  
- Remove items from the list  
- Enjoy a sleek dark blue and black theme  

## Technologies Used
- Vue 3 (CDN)
- Font Awesome (CDN)
- HTML5
- CSS3
